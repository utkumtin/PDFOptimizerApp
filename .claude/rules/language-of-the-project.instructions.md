## Language — English Only

**Everything in the codebase must be written in English. No exceptions.**

This includes:

- **Function, variable, class, and file names** — `kullaniciAdi` ❌ → `username` ✅
- **Comments** — inline, block, and JSDoc/TSDoc
- **Commit messages**
- **README and all documentation files**
- **Error messages and log output** — these end up in terminals and bug trackers, always English
- **Branch names and PR descriptions**
- **TODO / FIXME / NOTE annotations**

### Why this matters

A codebase is a shared artifact. The moment a non-English word slips into a function name or comment, you've silently locked out every developer who doesn't speak that language — including your future self using a search engine, Stack Overflow, or an AI assistant that reasons better in English.

English is the lingua franca of code. Embrace it fully.
